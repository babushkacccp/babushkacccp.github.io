По обращениям есть план в программе гос гарантий (раздел 7).

План по количеству и сумме.

Регионы, и МО, соотвественно должны выполнить этот план.

Поэтому при планировании идут путем тарификации обращений, так как при тарификации по услугам можно не попасть в стоимость обращения.

 

Обращения:

- по заболеванию (одна специальность врача или взаимозаменяемые специальности (терапевт, врач общей практики) + ID "Истоии болезни АПП"

- реабилитации. конечный набор посещений (услуг) + ID "Истоии болезни АПП"

 

Комплексные посещения:

- Диспансеризации

- Профилактические осмотры

 

Отдельный вопрос по посещениям по "Неотложной помощи" (планируются и тарифицируются по посещению)

- как тарифицировать дальнейшее обращение?

- включать ли посещение по неотложной помощи в обращение, как начало лечения.

Скоре всего надо просто разделить и не вклюючать. Если будет продолжено лечение, то будет открыто оборащение.
